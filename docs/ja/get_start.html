始めましょう

使用法

  1. アドオンをインストールすると、N パネル -> 編集 -> Easy Patch にボタンが表示されます。(編集モードでのみ表示されます)

  2. スナップしたいオブジェクトを選択します。

  3. スタートボタンをクリックします。

  4. マウスをクリックしてドラッグし、境界を描きます。(頂点にスナップするには Ctrl キーを押します)

  5. 境界がループを形成した後、アドオンはパッチを生成します。

  6. メッシュを適用するには Enter キーを押します。
